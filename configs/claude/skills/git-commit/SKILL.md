# Commit Discipline

## Commit Message Format

```
<type>: <concise description>

[optional body with details]
```

| Prefix      | Description       |
| ----------- | ----------------- |
| `refactor:` | Structural change |
| `feat:`     | New functionality |
| `fix:`      | Bug fix           |
| `chore:`    | Maintenance       |
| `docs:`     | Documentation     |

Small, frequent commits.
